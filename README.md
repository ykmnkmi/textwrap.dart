Text wrapping and filling. It's a pure port of [textwrap][textwrap] from Python.

## Usage

A simple usage example:

```dart
import 'package:textwrap/textwrap.dart';

void main() {
  print(fill('Text wrapping and filling.', width: 16));
  // Text wrapping
  // and filling.
}
```
